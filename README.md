# スタレゾMaxroll翻訳（英語 → 日本語）

「スターレゾナンス（Star Resonance / BPSR）」の **Maxroll ビルドガイド**を日本語化するブラウザ拡張です。
Maxrollをそれなりに自然な形で翻訳します。（**英語→日本語専用**）

- **対象は `maxroll.gg/blue-protocol/` 配下のみ**。他ゲームのページでは動作しません。
- 動作モードは「**自動**（開くと翻訳）／**手動**（ボタン・Alt+T）」を選べます。
- 端末内AIの翻訳のため、文法はやや不自然になることがあります。

---

## 対応ブラウザ

| ブラウザ | 対応 | 備考 |
|---|---|---|
| **Chrome（デスクトップ・138以降）** | ✅ 推奨 | Chrome内蔵翻訳APIを使用。初回のみ翻訳モデルをDL |
| **Edge** | ❌ 非対応 | 内蔵翻訳APIが使えず**正しく動作しません** |
| **Firefox** | ❌ 非対応 | 

> このツールは Chrome の内蔵翻訳で翻訳します。

---

## インストール（Chrome）

1. [Releases](https://github.com/pupupue-3/bpsr-translate-extension/releases) から `bpsr-translate-chrome.zip` をDLして解凍
2. `chrome://extensions` を開く
3. 右上の「**デベロッパーモード**」を ON
4. 「**パッケージ化されていない拡張機能を読み込む**」→ 解凍した **bpsr-translate-chrome** フォルダを選択
5. ツールバーにアイコンが出れば完了

---

## 使い方

1. `maxroll.gg/blue-protocol/` を開く（スタレゾ関連であれば別のページでも可）
2. **初回のみ**、画面上部に「▶ クリックで翻訳を有効化（初回のみモデルDL）」のバナーが出るので**クリック**
   - 翻訳モデルがダウンロードされます。
3. 2回目以降は自動で翻訳されます（既定は自動モード）

**手動で翻訳**：拡張アイコン →「▶ このページを今すぐ翻訳」、またはショートカット **Alt+T**。

### 設定（拡張アイコンをクリック）

| 項目 | 説明 |
|---|---|
| 拡張を有効にする | ON/OFF |
| 動作モード | 「自動」=開くと翻訳（既定）／「手動」=ボタン・Alt+Tで発動 |
